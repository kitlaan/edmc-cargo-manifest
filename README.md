# EDMC Cargo Manifest

Elite: Dangerous Market Connector (EDMC) plugin to display information about
the cargo inventory.

## Notes

Requires Python 3.11+ (EDMC 5.7.0+).
Developed starting with EDMC 5.13.0, so YMMV.
